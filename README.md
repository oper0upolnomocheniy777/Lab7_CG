# 3D Model Viewer - Программа для работы с 3D графикой

## Назначение программы

Программа предназначена для работы с трехмерной графикой и предоставляет следующие возможности:

### Основные функции:

1. **Загрузка и сохранение 3D моделей**
   - Поддержка формата Wavefront OBJ
   - Отображение моделей с гранями и ребрами
   - Совместимость с моделями из Blender и других 3D редакторов

2. **Создание фигур вращения**
   - Задание образующей кривой набором точек
   - Вращение вокруг осей X, Y, Z
   - Настройка количества сегментов (разбиений)
   - Автоматическое построение треугольных граней

3. **Построение графиков функций двух переменных**
   - Задание функции в формате z = f(x, y)
   - Настройка диапазонов по осям X и Y
   - Задание количества разбиений сетки
   - Поддержка математических функций через numpy и math

4. **Аффинные преобразования**
   - Перемещение (трансляция) по осям
   - Поворот вокруг осей X, Y, Z
   - Масштабирование по осям

### Особенности реализации:

- **Русскоязычный интерфейс** - все элементы управления на русском языке
- **Автоматическое масштабирование** - автоматическая настройка обзора под загруженную модель
- **Расширенная поддержка OBJ** - обработка сложных форматов с текстурными координатами и нормалями
- **Триангуляция полигонов** - автоматическое преобразование N-угольников в треугольники
- **Визуализация в реальном времени** - мгновенное отображение изменений

### Технические требования:

- Python 3.6+
- Библиотеки: numpy, matplotlib, tkinter

### Использование:

1. **Загрузка модели**: Используйте кнопку "Загрузить OBJ модель" для открытия файлов
2. **Создание фигуры вращения**: 
   - Задайте точки образующей в формате "x,y"
   - Выберите ось вращения
   - Укажите количество сегментов
3. **Построение графика функции**:
   - Введите функцию f(x,y)
   - Задайте диапазоны для X и Y
   - Укажите количество разбиений
4. **Преобразования**: Используйте панель аффинных преобразований для перемещения, поворота и масштабирования

Программа идеально подходит для учебных целей и демонстрации основ 3D графики и математического моделирования.
