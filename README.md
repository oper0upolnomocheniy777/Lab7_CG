# 3D Geometry Processor

Проект для работы с 3D-моделями, включающий загрузку/сохранение OBJ-файлов, построение фигур вращения и поверхностей функций.

## Функциональность

### 1. Работа с OBJ-файлами
- **Загрузка** моделей многогранников из файлов формата Wavefront OBJ
- **Сохранение** моделей в OBJ-формат
- **Визуализация** загруженных моделей
- **Аффинные преобразования** для моделей (перемещение, масштабирование, вращение)

### 2. Построение фигур вращения
- **Задание образующей** через набор точек
- **Выбор оси вращения** (X, Y, Z)
- **Указание количества разбиений**
- **Автоматическое вычисление** угла вращения (360°/N)
- **Построение модели** с данными о гранях
- *Опционально: сохранение и загрузка полученных моделей*

### 3. Построение графиков функций двух переменных
- **Задание функции** вида f(x, y) = z
- **Определение диапазонов**: [x₀, x₁] и [y₀, y₁]
- **Указание количества разбиений** по осям
- **Построение сегмента поверхности** с данными о гранях
- **Визуализация** и **аффинные преобразования**
- *Опционально: сохранение и загрузка моделей*

## Использование

### Требования
- Python 3.x
- Библиотеки: PyOpenGL, PyQt5/PySide2, NumPy

### Запуск
```bash
python main.py
```

### Основные операции
1. **Файл → Открыть** - загрузка OBJ-модели
2. **Файл → Сохранить** - сохранение текущей модели
3. **Построение → Фигура вращения** - создание фигуры вращения
4. **Построение → Поверхность функции** - построение графика функции
5. **Мышь + клавиши** - навигация и преобразования в 3D-пространстве

## Форматы файлов

### Wavefront OBJ
Поддержка стандартного формата OBJ с вершинами (v), нормалями (vn) и гранями (f).

Пример структуры:
```
v 0.0 0.0 0.0
v 1.0 0.0 0.0
v 0.0 1.0 0.0
f 1 2 3
```

## Примеры использования

### Фигура вращения
- Образующая: массив точек [(0,0,0), (1,0,0), (1,1,0)]
- Ось вращения: Y
- Количество разбиений: 36
- Результат: коническая поверхность

### Поверхность функции
- Функция: f(x,y) = sin(x) + cos(y)
- Диапазон: x∈[-π,π], y∈[-π,π]
- Разбиения: 50×50
- Результат: волнистая поверхность
